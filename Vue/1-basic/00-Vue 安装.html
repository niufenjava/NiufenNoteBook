Vue 安装

1. Vue Devtools

在开发 Vue 时，可以在浏览器上安装 Vue Devtools，方便Vue的调试。

Vue Devtools

2. 使用 script 标签引入 Vue

直接下载并使用 script 标签引入，Vue 会被注册为一个全局变量。

注：在开发环境下不要使用压缩版本，不然就会失去所有常见错误相关的警告

开发版本：https://vuejs.org/js/vue.js 包含完整的警告和调试模式
生产版本：https://vuejs.org/js/vue.min.js 删除了警告，33.30KB min+gzip

2.1 CDN

  • 对于制作原型或学习，可以使用如下方式：
    • https://cdn.jsdelivr.net/npm/vue
  • 对于生产环境，推荐链接到一个明确的版本号和构建文件，避免新版本造成不可预期的破坏
    • https://cdn.jsdelivr.net/npm/vue@2.6.10/dist/vue.js

3. NPM 方式

在用 Vue 构建大型应用时推荐使用 NPM 安装。NPM 能够很好地和 webpack 模块打包器配合使用。

npm install vue

4. 命令行工具（CLI）

Vue 提供了官方的 CLI，为单页应用（SPA）快速搭建复杂的脚手架。

vue-cli github vue-cli 文档
